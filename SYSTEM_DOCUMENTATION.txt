1. SYSTEM OVERVIEW
-----------------
The DIGI_MC system is a comprehensive digital document management solution
that provides secure storage, retrieval, and management of official documents
with both local and cloud-based storage capabilities.

2. CORE FUNCTIONALITIES
----------------------
2.1 Document Management
    ✓ PDF document upload and storage
    ✓ Automatic Google Drive backup
    ✓ Document categorization
    ✓ Version control
    ✓ Audit trail tracking
    ✓ Local and cloud synchronization

2.2 User Management
    ✓ Role-based access (Admin/User)
    ✓ User registration workflow
    ✓ Privilege management
    ✓ Profile administration

2.3 Search System
    ✓ Advanced document search
    ✓ Category/folder filtering
    ✓ Full-text search
    ✓ Date/title/case number filters

2.4 Dashboard Features
    ✓ Personalized user dashboards
    ✓ Document preview
    ✓ Download functionality
    ✓ Real-time notifications
