│                           HOW TO TEST NOW                                    │
└──────────────────────────────────────────────────────────────────────────────┘

Step 1: Make sure your server is running
        If not, run: php artisan serve

Step 2: Open your browser and log in as a REGULAR USER
        (Not admin - chatbot only appears for regular users)

Step 3: Look at the lower right corner
        You should see: 🟣 (a purple circular button with chat icon)

Step 4: Click the button
        The chat window will slide up smoothly

Step 5: Type a message
        Try: "Hello" or "Help me with documents"

Step 6: Press Enter or click the send button
        The bot will respond instantly!

│                         TROUBLESHOOTING                                      │
└──────────────────────────────────────────────────────────────────────────────┘

Problem: Chatbot doesn't appear
Solution: 
  - Make sure you're logged in as a USER (not admin)
  - Hard refresh the page (Ctrl + F5)
  - Check browser console (F12) for errors

Problem: Chatbot appears but doesn't respond
Solution:
  - Check Network tab in DevTools (F12)
  - Look for POST to /api/dialogflow
  - Check storage/logs/laravel.log

Problem: Styling looks off
Solution:
  - Clear browser cache
  - Hard refresh the page
  - Check if CSS is loading properly
